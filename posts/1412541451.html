日语动词变形

发表于2025-08-20|更新于2025-08-21|日语文法
|总字数:256|阅读时长:1分钟|浏览量:

ます形

一类 二类 する くる
う段 → い段 + ます る → ます します きます

て形

一类 二类 する くる
う、つ、る → って る → て して きて
む、ぶ、ぬ → んで - - -
く / ぐ → いて / いで - - -
す → して - - -
行く → 行って - - -

た形

一类 二类 する くる
う、つ、る → った る → た した きた
む、ぶ、ぬ → んだ - - -
く / ぐ → いた / いだ - - -
す → した - - -
行く → 行った - - -

ない形

一类 二类 する くる
う段 → あ段 + ない る → ない しない こない
う → わ + ない - - -
ある → ない - - -

命令形

一类 二类 する くる
う段 → え段 る → ろ しろ こい

意志形

一类 二类 する くる
う段 → お段 + う る → よう しよう こよう

假定形

一类 二类 する くる
う段 → え段 + ば る → れば すれば くれば

可能形(二类)

一类 二类 する くる
う段 → え段 + る る → られる できる こられる

被动形(二类)

一类 二类 する くる
う段 → あ段 + れる る → られる される こられる
う → わ + れる - - -

使役被动形(二类)

在使役上变被动，一类约音。

一类 二类 する くる
う段 → あ段 + される(せられる) る → させられる させられる こさせられる
う → わ + される(せられる) - - -

使役形(二类)

一类 二类 する くる
う段 → あ段 + せる る → させる させる こさせる
う → わ + せる - - -
